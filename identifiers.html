Identifiers

  • Identifiers are nothing but names of variables,functions,constants

    • Rules:

  • Identifiers should contains alphabet,numbers,dollar sign,underscore
  • It should be start with alphabet or underscore
  • It should not be start with number and dollar sign
  • It should not contain any special characters except dollar sign and underscore
  • Identifier name should not be any keyword
  • It is a case sensitive
